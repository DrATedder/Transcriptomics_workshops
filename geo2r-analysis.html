Differential Expression with GEO2R

Purpose: Perform exploratory analysis of gene expression data without coding.

1. Introduction and Rationale

GEO2R is a web-based tool from the NCBI's Gene Expression Omnibus (GEO) platform, allowing users to identify differentially expressed genes between groups. It is designed for users with minimal coding experience and supports microarray datasets.

This tool compares gene expression levels across defined sample groups (e.g., treated vs. control) to find statistically significant changes in gene expression.

GEO2R Website: https://www.ncbi.nlm.nih.gov/geo/geo2r/

2. Understanding the Data

GEO Series and Samples

GEO Series (GSE IDs) are experimental datasets that group individual samples (GSMs). They are associated with a platform (GPL) which provides information about the measurement technology used.

Platforms and Probes

Each GEO Series is linked to a platform defining probe sets, such as Affymetrix or Illumina chips. Each probe corresponds to a gene or transcript.

3. Preparing for Analysis

Choose a Dataset
  • Visit https://www.ncbi.nlm.nih.gov/gds
  • Search with keywords (e.g., “liver cancer”)
  • Ensure at least 3 biological replicates per condition
Examine Study Design

Check the study metadata for grouping variables (treatment/control, time point, etc.). Proper group assignment is essential for valid statistical inference.

4. Performing the Analysis

Step 1: Access GEO2R
  • Go to the chosen GEO Series page (e.g., GSE12345)
  • Click "Analyze with GEO2R"
Step 2: Define Sample Groups

Assign each sample to Group 1 or Group 2 by clicking its label. Base this on the original study’s metadata.

Step 3: Customize Options
  • Log Transformation: Leave ON unless otherwise noted
  • Statistical Test: GEO2R uses Limma (moderated t-test)
Step 4: Run the Analysis

Click “Top 250” for a quick view or “Save All Results” to download the full dataset.

5. Interpreting Results

ColumnDescription
Gene.symbolOfficial gene symbol
adj.P.ValAdjusted p-value (FDR control)
P.ValueUnadjusted p-value
logFCLog2 fold-change
tt-statistic (from Limma)
BLog-odds of DE
Use adj.P.Val < 0.05 and |logFC| ≥ 1 for strong candidate genes.

6. Multiple Testing and Volcano Plot

GEO2R applies the Benjamini-Hochberg procedure to control the False Discovery Rate. This reduces false positives due to multiple comparisons.

Volcano Plot (Optional)

Plot log2 Fold Change vs. –log10 Adjusted p-value to visualize effect size and significance.

7. Exporting and Downstream Analysis

Click “Save All Results” to export. Open in Excel or import into R/Python for additional filtering or visualization.
